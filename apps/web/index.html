How it works

Plain English in. On-chain settlement out.

Type a deal in natural language. An LLM parses it into a validated EscrowSpec. You confirm. Funds enter EscrowBook on Base Sepolia. The counterparty submits an evidence hash. You release — or the contract auto-refunds after the deadline.

Security boundary: the LLM never decides who signs. The payer is always msg.sender. Prompt-injection attempts are flagged in warnings; the strict JSON schema rejects unknown fields.
